스프링 Bean 클래스 이름 오류(postfix *Listener)

스프링 Bean 클래스 이름이 "Listener"로 끝날 경우 생기는 문제점
  • October 27, 2021
  • springboot  
  • spring  
  • bean  
문제 스프링부트 환경(version: 2.5.x)에서 Bean 클래스 작성 시 클래스 이름이 Listener 로 끝날 경우 bean 생성 중 에러가 발생한다. incorrect example 아래의 예제처럼 클래스 이름이 Listener 로 끝날 경우 bean 생성이 정상적으로 이루어지지 않는다. 1 2 3 4 @Service class CustomEventListener { } correct example 1 2 3 4 @Service class CustomEventReceiver { } 위와 같이 클래스 이름을 Receiver 로 변경해주면 bean 생성이 정상적으로 마무리 된다.
